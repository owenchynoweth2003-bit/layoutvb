# TileVision Pro
Professional tile layout planning application built with React + Vite.

## Features
- Multi-wall tile layout planning
- 15 tile patterns (running bond, herringbone, chevron, diagonal, etc.)
- Real-time 2D canvas preview with cut analysis
- Interactive 3D room visualization
- Custom tile image upload
- Custom polygon wall shapes
- Per-wall pattern support
- Wall visibility toggles
- Floor/ceiling surface support
- Export to PNG, clipboard report, and print

## Getting Started
```bash
npm install
npm run dev
```

## Deployment
Configured for Vercel deployment via `vercel.json`.

```bash
npm run build
```
